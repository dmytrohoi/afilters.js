Filter

new Filter($0: Object, options: Object, String: string?, Array: Array<Function>?)
Parameters
$0 (Object = {})
Name Description
$0.filters any (default [])
$0.approach any (default 'AND')
$0.final any
options (Object) The desired options to set
String (string?) options.approach - Pre-defined approach
Array (Array<Function>?) options.filters - Pre-defined list of filter function
Example
> const f = new Filter((arg1) => Boolean(arg1));
> f.try(1).finally((arg1, {returned}) => console.log(`Arg ${arg1} is ${returned[0]}`));
Arg 1 is true
> f.addFinal((arg1, {returned}) => console.log(`Arg ${arg1} = ${returned[0]}`));
> f.repeat(2);
Arg 2 = true
